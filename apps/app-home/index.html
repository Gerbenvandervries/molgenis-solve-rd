Get Started

Welcome to RD3 Solve-RD rare disease metadata database. The RD3 —or Rare Disease Data about Data— database provides an overview of Solve-RD "asset collections". This includes metadata and files associated with patients and samples, as well as an overview on the analyses that were performed.

There are several ways to get started. If you are familiar with the structure of RD3, you can view the tables using the Table Navigator. The Discovery Nexus tool can also be used to discover data within RD3. In addition, visit the "Get Started" page to find view data by release.
